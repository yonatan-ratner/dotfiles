# dotfiles

My .files manages with stow

## process


1. install GNU Stow using the OS' package manager, for example:

  - ```sudo apt install stow```
</br></br>

2. clone this repository and cd into it

  - ```git clone 'this repo' && cd 'this repo'```
</br></br>

3. activate stow

     stow everything:

   - ```stow *```

     or stow specific:

   - ```stow nvim```
   - ```stow zsh```
